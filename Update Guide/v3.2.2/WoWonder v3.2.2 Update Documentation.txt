1) How To Update

- BACKUP, BACKUP, AND BACKUP everything, files, DATABASE. Please BACKUP!

- Upload "update.php" file located on "Update Guide/v3.2.2" folder to your server.

- Upload and overwrite ALL files/folders (SELECT ALL) file(s) located inside "Update Guide/v3.2.2/Script/" folder to your server, if you have renamed your themes, please open "Update Guide/v3.2.2/Script/themes/wowonder/" and upload & overwrite ALL files/folders to your theme folder.

- Once the files are uploaded, open and run http://www.YOURSITE.COM/update.php and follow the instructions.

- Note: Don't run "update.php" file until all the files are uploaded.

- Note: If you are using nodejs, reboot your nodejs server. 

- For more information, please read our FAQ https://docs.wowonder.com/faqs.html#a-234234
